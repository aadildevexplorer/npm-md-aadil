# simple-slugify

A lightweight slugify package for Node.js.

## Installation

```bash
npm install simple-slugify
```

## Usage

```javascript
const slugify = require("simple-slugify");

console.log(slugify("Hello World"));
// hello-world

console.log(slugify("Node.js & Express"));
// nodejs-express

console.log(slugify("Mohammad Aadil Khan"));
// mohammad-aadil-khan
```
